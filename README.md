## 2. Interface Overview
The application is designed around a central grid view with collapsible panels.

![Main Grid View](https://photos.smugmug.com/photos/i-P8WMJvD/0/M/i-P8WMJvD-M.jpg)

### Top Toolbar
![Toolbar Detail](https://photos.smugmug.com/photos/i-CCmkVrW/0/M/i-CCmkVrW-M.jpg)

* **Profile:** Select the AI system prompt (e.g., "Stock Photography", "Social Media"). Click the **Gear icon** to edit profiles.
* **Run AI:** The green play button starts the analysis for selected images.
* **Speed:** Controls the number of parallel threads (1-20). Higher is faster but uses more CPU/Bandwidth.
* **Columns:** Adjust the number of columns in the grid or set to **"Auto"**.
* **Sorting:** Sort by Name or Date (Ascending/Descending).
* **Flat View:** Toggle to see all files from subfolders in a single flat list.
* **Backup:** Checkboxes to enable/disable backups for CSV, XMP, and Original files.

### Filter Bar
![Filter Bar Detail](https://photos.smugmug.com/photos/i-hCrtjrQ/0/M/i-hCrtjrQ-M.jpg)

Located below the toolbar, this bar helps you organize your workspace.
* **Select:** Buttons to Select All / Select None.
* **Search:** Filter images by text (Title, Description, Keywords, or Filename).
    * **Aa:** Toggle case sensitivity.
    * **Target:** Choose which field to search in.
* **Status Filter:** Filter by state (Selected, Modified, Done, Error, etc.).
* **Type Filter:** Show only RAW, JPG, or Video files.
* **Toggle Panels:** Buttons to show/hide the **Rating Bar** and **Batch Edit Bar**.

### The Image Grid
Each cell represents a media file.
* **Visual States:**
    * **Gray/White:** Unchanged / Default.
    * **Yellow:** Modified (unsaved changes).
    * **Green:** Saved / Metadata loaded.
    * **Red:** Validation error (e.g., title too long).
* **Badges:** Small icons on thumbnails provide quick info:
    * GPS: File has geolocation data.
    * **XMP:** An XMP sidecar file exists.
    * **ORIG:** An original backup exists.
    * **RAW:** Indicates a RAW file.
    * **VIDEO:** Indicates a video file.
    * ⚠️: Indicates a corrupted file or load error.

## 5. Batch Operations
Toggle the **Batch Edit** panel using the list icon in the filter bar. This allows you to modify hundreds of files at once.

1.  **Select Target:** Choose where to apply changes (Title, Description, Keywords, or Everywhere).
2.  **Input Text:** Type the text you want to add, remove, or find.
3.  **Actions:**
    * **Add:** Appends text. For keywords, it adds a new tag. For text, it appends with a space.
    * **Remove:** Removes the specified word or tag.
    * **Replace:** Replaces "Text A" with "Text B". Use format `Old->New` or open the dedicated Replace Dialog (`Ctrl+H`).
    * **Delete Field:** Clears the entire content of the selected field.
    * **Clear All:** Removes ALL metadata from selected files.

## 6. Professional Advanced Tools

### 🧠 Unlimited Customization with AI Profiles
Turn ArtushVision AI into your personal assistant. By using **Dynamic Variables**, you feed the AI technical and contextual data hidden in your files, allowing it to "understand" your photos far beyond just pixels.

* **👨‍👩‍👧 Family & Event Archiving:** Use **{folder_context}** to tell AI about specific people or celebrations based on your folder names.
    * *Example prompt:* "This photo is from {folder_context}. Identify the people and the event name."
* **🌍 AI Visual Geolocation:** Even without GPS, the AI recognizes world landmarks, cities, and specific streets purely by pixels (Google Lens style).
    * *Example prompt:* "Recognize the landmark visually. If unsure, use {loc_hint} as a starting point."
* **📝 Text & OCR Recognition:** Automatically extract text from signs, handwritten notes, or notices found directly within the image.
    * *Example prompt:* "Transcribe the text {text_ocr} from this image and add it to the description."
* **📍 Precise Geographic Integration:** Supply raw coordinates or map links to help the AI pinpoint exact buildings or mountain peaks with 100% accuracy.
    * *Example prompt:* "Identify the specific building at {gps_raw}. Context: {maps_link}"

> 🛡️ **Smart Metadata Protection:** Your manual work is safe. ArtushVision AI **preserves all existing keywords and Face Recognition (People tags)**. New AI insights are intelligently appended to your current metadata, never overwriting your valuable tags.

#### Available Variables:
| Variable | Description | Example Output |
| :--- | :--- | :--- |
| `{text_ocr}` | Text extracted visually from the photo (signs, labels). | "Route 66 - Historic Landmark" |
| `{folder_context}` | The name of the parent folder (for personal context). | "Grandma's 80th Birthday" |
| `{loc_hint}` | City and Country identified from coordinates or pixels. | "Prague, Czech Republic" |
| `{date_info}` | The exact capture date from metadata (DD.MM.YYYY). | "24.12.2025" |
| `{gps_raw}` | Raw GPS coordinates for precise technical mapping. | "50.08, 14.43" |
| `{maps_link}` | A direct link to Google Maps for the specific location. | http://maps.google.com/... |

> 💡 **Pro Tip:** Combine variables: *"This photo from {folder_context} was taken in {loc_hint}. Identify all landmarks and transcribe any text found: {text_ocr}."*
