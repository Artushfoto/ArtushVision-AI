OpenRouter API Key Setup Guide for Beginners
Step-by-step guide to register and generate your secure API key.

Go to the Website: Navigate to the OpenRouter platform.
Create Your Account: Sign up using your email address and a chosen password.
Confirm Your Email: Check your inbox and confirm your email address as instructed in the confirmation email.
Create the API Key: Navigate to the API key section and click "Create". Set a name, for example: ArtushVision AI.
