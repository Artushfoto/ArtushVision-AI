1. Installation & Setup

System Requirements

  • OS: Windows 10/11 (64-bit).
  • Internet: Required for AI analysis and maps.
  • Dependencies: ExifTool and FFmpeg (bundled with the app).

First Launch

  1. Run ArtushVisionAI.exe.
  2. Enter your License Key (or use Trial).
  3. Set your OpenRouter API Key in File > API Key.

2. Interface Overview

Top Toolbar & Grid

  • Profile: Select AI prompts (Stock, Travel, etc.).
  • Run AI: The green play button starts analysis.
  • Image Grid: Color-coded states (Yellow = Modified, Green = Saved).

3. AI Analysis Workflow

  1. Load Folders: Select your directory.
  2. Select Files: Click or use Shift+Click.
  3. Run AI: Real-time cost feedback is provided per query.
