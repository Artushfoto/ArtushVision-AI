# 🚀 OpenRouter Setup Guide

Follow these steps to register and generate your API key for this project.

---

## 1. Create an Account
1. Go to [OpenRouter.ai](https://openrouter.ai/).
2. Click on the **Login** or **Sign Up** button in the top right corner.
3. You can sign up quickly using your **Google account**, **GitHub account**, or an **Email**.

## 2. Generate Your API Key
1. Navigate to the [Keys section](https://openrouter.ai/settings/keys) in your dashboard.
2. Click the **"Create Key"** button.
3. **Name your key** (e.g., `My-Project-App`) so you can identify it later.
4. **Copy the Key immediately.** > ⚠️ **Important:** For security reasons, you will only see this key **once**. Copy it and store it in a safe place.

## 3. Add Credits (Optional)
* OpenRouter offers both **free** and **paid** models. 
* To use paid models (like GPT-4 or Claude 3), go to **Credits** in the sidebar and add a small balance.
* Many models are labeled as **Free** and do not require a balance.
