# 📺 ArtushVision AI – Video Tutorials & Documentation

Welcome to the official workflow guide for **ArtushVision AI**. This document provides a structured overview of all features, from basic interface setup to advanced AI integration.

---

## 01. Interface Overview
*Initial setup and navigating the application environment.*

- **Menu & Settings** – Global application configuration and preferences.
- **Tooltips** – Context-aware help for quick feature orientation.

> [!TIP]
> Use the tooltips to quickly understand icons without leaving your current workflow.

---

## 02. Working with Grid (Basic)
*Efficiently managing your asset library and view modes.*

- **Asset Import:**
  - Load Folder & Subfolders support.
  - Media filters: Raw, Video, or All files.
- **View Customization:**
  - Adjustable Column size & Keyword field height.
  - Fast grid scrolling for large libraries.
  - Flat view mode (ignore folder structure).
- **Sorting & Search:**
  - Sort by A-Z, Z-A, or Date of Original.
  - Global Search: Keywords, Description, and Title.
  - Toggle between **Text Mode** and **Tag Mode**.
  - UI Localization (e.g., Czech grid translation).

---

## 03. Translation and Synonyms
*Localizing metadata and expanding your reach.*

- **Multi-language Workflow:** Translate Title, Description, and Keywords.
- **Language Switcher:** Seamlessly move between English, German, Spanish, and Czech.
- **Synonym Integration:** Automatically expand metadata with related terms.

---

## 04. Synonyms and User Dictionary
*Customizing the AI and translation vocabulary.*

- **User Dictionary:** Add custom industry-specific terms.
- **Synonym Management:** Define custom relations for more accurate tagging.

---

## 05. Filter Limits and Modified Files
*Data validation and tracking changes.*

- **Advanced Filtering:**
  - Setup Max/Min/All limits for metadata length.
  - **Filter Exceed Limits:** Identify files that don't meet agency requirements.
- **Batch Editing:**
  - Edit Title/Text, add/remove keywords.
  - **Autocompleter:** Enable or disable smart suggestions.
  - **Modified Status:** Filter and save only changed assets.

---

## 06. Working with Keywords
*The core of ArtushVision AI metadata management.*

- **Manual Tagging:** Add keywords in Text or Tag mode.
- **Intuitive UI:** Drag and drop keywords between assets.
- **Bulk Operations:**
  - Bulk delete selected tags.
  - Copy/Paste keywords across multiple files.
  - **Bulk Add:** Insert keywords at the beginning or end of the list.

---

## 07. Map Function – Ratings – EXIF
*Geodata and technical metadata management.*

- **Geographic Tools:** Google Maps and OpenStreetMap integration.
- **Technical Data:** Full EXIF metadata viewer.
- **Rating System:**
  - Set star ratings directly in the Grid or Edit window.
  - Filter assets based on rating values.

---

## 08. AI Functions
*Harnessing the power of Large Language Models.*

- **Prompt Editor:** Select your preferred AI model and customize instructions.
- **Grid AI:** Batch process multiple images/videos using AI.
- **Edit Window AI:** Deep analysis and captioning for individual assets.
- **Monitoring:** Track your usage with OpenRouter statistics.
