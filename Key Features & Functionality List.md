### Key Features & Functionality List

* **Versatile AI Engine Modes**
    * **Cloud AI:** High-speed analysis via OpenRouter (Gemini 2.0 Flash).
    * **Local AI:** 100% private offline analysis via Ollama.
    * **Hybrid AI:** Local vision analysis combined with cloud-based text formatting.
    * **2-Pass Local AI:** Deep offline synthesis using a dedicated Vision Model followed by a Text Model pass.
* **Getty Images Optimizer (Master ESP Tool)**
    * Non-destructive workflow with `.getty` sidecar files.
    * Interactive Getty Resolver (Single & Batch modes).
    * Master Dictionary with 11,746+ commercial terms.
    * Semantic AI Disambiguation for homonyms.
    * Visual context highlighting and term splitting.
* **Global Distribution & Smart FTP**
    * Multi-server batch uploads with per-server threading.
    * Universal Category Mapping (Adobe Stock, Shutterstock, etc.).
    * Automated per-agency CSV metadata generation.
    * FTP Status Badges written directly into XMP metadata.
    * Custom FTP Profiles (e.g., "Stock Photo", "Editorial").
* **Advanced Interface & Organization**
    * **Top Toolbar:** AI Profiles, Absolute Priority AI Hints, and Speed/Thread control.
    * **Filter Bar:** Smart Search (Aa/Case sensitivity), Folder Filtering with photo counts, and File Type Filters.
    * **Image Grid:** Visual Status Indicators (Gray/Yellow/Green/Red) and Thumbnail Badges (GPS, XMP, ORIG, RAW, VIDEO, GETTY, FTP, CAT).
    * **Flat View:** Unified list view for all subfolders.
* **Professional Metadata Management**
    * **Detail Photo View:** Inline editing with real-time word/character counters.
    * **Keyword Bubbles:** Color-coded (AI, Manual, Original) with drag-and-drop reordering.
    * **Smart Metadata Protection:** Preservation of existing keywords and Face Recognition (People tags).
    * **Auto-GPS Country Lookup:** Reverse geocoding via OpenStreetMap or ArcGIS.
    * **Multilingual Spell Check:** Real-time underlining and suggestions.
* **Powerful Batch Operations**
    * **Smarter Batch Rename:** Dynamic variables (`{TITLE}`, `{CC}`, `{DATE}`, `{FOLDER_NAME}`).
    * **Bulk Metadata Editing:** Smart Add/Remove/Replace with `Old -> New` syntax and `Ctrl+H` support.
    * **File Operations:** Integrated Copy, Move, and Delete (with sidecar synchronization).
* **Comprehensive Settings & Configuration**
    * Ollama VRAM Context tuning.
    * Video Economy Mode (Collage-based analysis).
    * Integrated Metadata Translator.
    * Field constraints and visual error feedback.
    * Export/Import for configuration backup.
