### 🌟 Key Features & Functionality

*   🤖 **Versatile AI Engine Modes**
    *   **Local AI (100% Free & Private):** Run standard visual analysis on your local hardware via Ollama. Perfect for sensitive shoots (boudoir, unreleased commercial products) with zero API costs.
    *   **Hybrid AI (Local Vision + Cloud Text):** Your GPU freely performs the heavy visual analysis, and a cloud model structures the text. Peak SEO quality at wholesale pricing.
    *   **2-Pass Local AI:** Deep offline synthesis. A local Vision model reads the pixels, then a second specialized text model formats a perfect JSON completely offline.
    *   **Cloud AI:** High-speed cloud processing via OpenRouter (Gemini 2.5 Flash) at raw cost.

*   🎯 **Getty Images Optimizer (Master ESP Tool)**
    *   **Near-Perfect Acceptance:** Map your tags against a built-in Master Dictionary of 9,867++ approved commercial terms.
    *   **Semantic AI Disambiguation:** The AI analyzes image context to solve homonyms (e.g., automatically distinguishing a "crane" bird from a "crane" construction machine).
    *   **Non-Destructive Workflow:** Original metadata remains untouched; optimized terms are safely stored in separate `.getty` sidecars.

*   📦 **Global Distribution & Smart FTP Suite**
    *   **Per-Server Multi-Threading:** Configure individual thread limits per agency to prevent connection blocks (`421 Too many connections`).
    *   **1-Click Automated CSV Upload:** The app generates a temporary, agency-specific CSV with matching category IDs, uploads it alongside your media, and automatically purges it from your drive.
    *   **FTP Status Badges:** Successful uploads write micro-badges directly into XMP metadata for persistent tracking.

*   🖥️ **Advanced Interface & Portfolio Organization**
    *   **Blazing-Fast Image Grid:** Displays Titles, Descriptions, and Keywords directly below every card. Real-time visual status states (Gray/Yellow/Green/Red) warn you about validation errors.
    *   **Absolute Priority AI Hints:** Force the AI to treat your manual hints (obscure locations, Latin species) as absolute facts, completely eliminating hallucinations.
    *   **Flat View:** Toggle a unified, continuous list view for all loaded subfolders simultaneously.

*   🛠️ **Professional Batch Operations & Linguistics**
    *   **Smart Append Sync:** Type a keyword into a selection of 500 photos—the app appends it in real-time without overwriting unique tags.
    *   **Dynamic Batch Rename:** Rename thousands of files using variables like `{TITLE}`, `{CC}`, `{DATE}`, or `{FOLDER_NAME}`.
    *   **Type-Ahead Autocomplete:** Predictive text powered by a pre-warmed database of 300,000+ highly commercial stock terms.
    *   **Dual-Language Spellcheck:** Validate against two dictionaries simultaneously (e.g., English + your native language) without switching settings.
    *   **Reverse Geocoding:** Auto-translates GPS coordinates into readable city/country names via OpenStreetMap or ArcGIS to give the AI geographical context.
